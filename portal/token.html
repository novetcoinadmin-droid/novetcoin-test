NVT公式トークン情報

NovetCoin / NVTについて
NVTは、創作投稿Webサービス「Novet」に関連して利用される独自トークンです。 投稿報酬、有料作品の購入、外部ウォレットとの連携など、創作活動を支援するための仕組みとして導入しています。
重要：
NVTは投資商品ではありません。価格上昇、利益、換金性、流動性、外部スワップ、買取、将来価値を保証するものではありません。 外部ウォレットやブロックチェーンを利用する場合は、利用者自身の判断と責任で管理してください。

2. NVTの目的

NVTは、Novet内で創作活動に対する価値体験を提供するために導入しています。 作品投稿、有料作品、報酬、外部ウォレット連携などを通じて、クリエイターの活動を支援することを目的としています。

Novetにおける価値の中心は、トークン価格ではなく、サービスの品質、投稿体験、読者とのつながり、 AIイラストや保管庫などの創作支援機能、そして作品そのものにあります。

3. NVTでできること

  • Novet内での投稿報酬として受け取る
  • 有料作品の購入に利用する
  • ユーザーのDynamicウォレットで保有する
  • 外部ウォレットへ出金する
  • Novet内でNPを買う機能に利用する

利用できる機能は、サービスの公開状況、メンテナンス、アカウント状態、ウォレット状態、ネットワーク状況により制限される場合があります。

4. MetaMaskへ手動追加する方法

追加前の確認：
MetaMaskのネットワークが Polygon Mainnet になっていることを確認してください。
  1. MetaMaskを開きます。
  2. ネットワークを「Polygon Mainnet」に切り替えます。
  3. 「トークンをインポート」または「Import tokens」を選びます。
  4. トークンコントラクトアドレスに、下記の公式アドレスを入力します。
  5. トークン名、シンボル、Decimalsが自動入力または手動入力されることを確認します。
  6. 内容を確認して追加します。

6. 詐欺トークン・偽トークンに注意

ブロックチェーン上では、第三者が似た名前のトークンを作成できる場合があります。 「Novet」「NovetCoin」「NVT」などに似た名称であっても、公式コントラクトアドレスと一致しないものはNovet公式トークンではありません。

  • 必ず公式ページのコントラクトアドレスを確認してください。
  • 知らない相手から送られたトークンやリンクを安易に操作しないでください。
  • 秘密鍵、シードフレーズ、リカバリーフレーズを誰にも教えないでください。
  • 外部サイトでウォレット接続する場合は、URLと権限内容を確認してください。
  • 価格や利益を保証する案内、購入を急がせる案内には注意してください。

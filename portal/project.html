1. Project Overview

Novet is a creative posting platform for novels, illustration-based works, and creator-driven digital content. The platform is designed to support creators by allowing them to publish works, manage creative assets, access paid-work features, and receive platform-based rewards.

NovetCoin / NVT is a utility token used in the Novet ecosystem. It may be used for creator rewards, paid creative works, purchasing Novet Points (NP), and wallet-related deposit or withdrawal functions.

Project NameNovet / NovetCoin
Project CategoryCreative Platform / Web3 / Creator Economy / Utility Token
Primary Platformhttps://novetcoin.com/
Main Use CaseCreator rewards, paid creative works, platform utility, NP purchase, and wallet functions.
